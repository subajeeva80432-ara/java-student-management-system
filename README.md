# java-student-management-system
A console-based Student Management System built using Java. This project demonstrates Object-Oriented Programming concepts such as classes, constructors, methods, conditional logic, and user input handling.

# Java Student Management System

## Features
- Input student details
- Store data using constructor
- Display student information
- Evaluate performance (Topper / Average)

## Concepts Used
- Classes & Objects
- Constructor
- Conditional Statements (if-else)
- Scanner for user input
- Basic OOP principles

## Technologies
- Java
- VS Code
